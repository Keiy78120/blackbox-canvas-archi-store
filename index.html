BLACKBOX Store

Plateforme B2B de réassort des salons. KHube (Office D-Studio) pilote l'interface, Shopify est le moteur caché — pas de checkout Shopify côté Store (paiement Stripe orchestré par KHube). 26 écrans en 3 zones.

Parcours principal (gérant)

  • A1 login → A3 mandat (1ʳᵉ fois) → B1 mes salons → B2 choisir salon → B3 catalogue → B5 panier → B6 checkout → B7 confirmation → B9 suivi.
  • Réassort récurrent : B1 → B10 (recommander en 1 clic).

Ce qui fait varier le compte (à border avec Ayoub)

  • Admin comptes/salons (3 écrans) réintégrés seulement si Inside n'est pas prêt.
  • États empty / loading / erreur + variantes mobile → doublent vite le nombre de frames.
  • SAV (B11) & tracking (B9) : version simple V1 ou riche selon le cadrage.
